## Technologien

-   [ReactJS](https://reactjs.org/)
-   [Redux](https://redux.js.org/)
-   [Material-UI](https://material-ui.com/)
-   [NestJS](https://nestjs.com/)
-   [GraphQL](https://graphql.org/)

## Frontend starten

`cd frontend`

`npm run start`
